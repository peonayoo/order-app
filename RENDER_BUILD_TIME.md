# Render.com 빌드 시간 가이드

## ⏱️ 일반적인 빌드 시간

### Static Site (프런트엔드)
- **첫 배포**: 2-5분
- **일반 빌드**: 1-3분
- **재배포**: 1-2분

### Web Service (백엔드)
- **첫 배포**: 3-5분
- **일반 빌드**: 2-4분
- **재배포**: 2-3분

## 📊 빌드 단계별 시간

### 프런트엔드 빌드 과정
1. **의존성 설치** (`npm install`): 30초 - 1분
2. **빌드 실행** (`npm run build`): 30초 - 1분
3. **파일 업로드**: 10-30초
4. **총 시간**: 1-3분

## ✅ 정상적인 빌드 시간

- **1분**: 정상 (의존성 설치 중)
- **2분**: 정상 (빌드 실행 중)
- **3분**: 정상 (거의 완료)
- **5분 이상**: 확인 필요

## 🔍 빌드 상태 확인 방법

### Render.com 대시보드에서 확인
1. 배포 중인 서비스 클릭
2. **"Logs"** 탭 클릭
3. 실시간 빌드 로그 확인

### 정상적인 로그 메시지
```
Installing dependencies...
Building for production...
Build completed successfully
```

### 문제가 있는 경우
- 에러 메시지 표시
- 빌드가 멈춤
- 5분 이상 진행되지 않음

## 🐛 빌드가 오래 걸리는 경우

### 가능한 원인
1. **의존성 설치 중**: 정상 (처음에는 오래 걸림)
2. **네트워크 지연**: 일시적
3. **빌드 오류**: 로그 확인 필요

### 해결 방법
1. **로그 확인**: 에러 메시지 확인
2. **기다리기**: 5분까지는 정상
3. **재시도**: 문제가 있으면 재배포

## 💡 팁

- 첫 배포는 항상 더 오래 걸립니다
- `node_modules`가 많으면 설치 시간이 길어집니다
- React + Vite 빌드는 보통 1-2분이면 완료됩니다
